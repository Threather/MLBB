## ğŸ’¡ How to Use

### Starting the Tournament

1. Click on any Upper Bracket match to select a winner
2. Click on a player's name to mark them as the winner
3. The loser will automatically drop to the Lower Bracket
4. Winners advance to the next round

### Features

- **Interactive:** Click players to select winners
- **Visual Feedback:** Winners are highlighted in green, losers in gray
- **Lower Bracket:** Automatically appears when first match is completed
- **Auto-Save:** Tournament progress is saved to your browser
- **Reset Button:** Start over anytime with the reset button

### Double Elimination System

- **Upper Bracket:** Winners stay here until they lose
- **Lower Bracket:** First-time losers get a second chance
- **Elimination:** Lose twice = OUT of tournament
- **Grand Finals:** Upper Bracket winner vs Lower Bracket winner
- **Bracket Reset:** If Lower Bracket winner wins Grand Finals, play one more match
